# Mushroom Jumper

Мини-игра, которая отвечает на строчку из известного батла: «Почему нет игры, где ты типа грибок, и ты прыгаешь на супермарио». Теперь она есть: вы играете за пиксельного грибка, прыгающего по платформам и оглушающего незадачливого водопроводчика.

## Как играть
- Стрелки влево/вправо — движение.
- Стрелка вверх — прыжок.
- Прыгайте на водопроводчика сверху, чтобы заработать очки.
- Потеряли все жизни? Нажмите **R**, чтобы мгновенно перезапустить попытку.

## Запуск
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите игру в режиме разработки:
   ```bash
   npm run dev
   ```
   Откройте предложенный адрес (по умолчанию `http://localhost:8080`).
3. Соберите продакшен-версию:
   ```bash
   npm run build
   ```
